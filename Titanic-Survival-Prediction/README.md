# Titanic Survival Prediction

## Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Random Forest Classifier

## Dataset Features
- Passenger Class
- Gender
- Age
- Fare
- Family Information
- Embarkation Port

## Model Used
Random Forest Classifier

## Result
Achieved approximately 82% prediction accuracy.
